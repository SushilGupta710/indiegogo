## General info
https://www.indiegogo.com/projects/turbohub-world-s-fastest-ssd-multiport-adapter--2#/
This is a node.js web application clone of above link powered by Express that provides the main functions you'd expect from a website, such as content, login and register, an User Interface, etc.
	
## Technologies
Project is created with:
* Node version: 12.17.0
* Express version: 4.17.1
* Bootstrap version: 4.5.3
* Handle-Bars(hbs) version: 4.1.1

	
## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm start OR $ npm run dev

```
